MG Property Services

Website (Individual project)

MG Property Services was built for my final year practical web project. Soon to be released, it will compete with existing property services companies located in the Bicester area.

The site's objectives were for it to be fully responsive, visually appealing and user-friendly. It involved finding my own client and working on a real project individually along with a 4000 word written report.

The site was built following the waterfall methodology and from conception to completion, involved website development tools and techniques such as competitor analysis , user testing and hi-fidelity wireframes.

The project was successful and the client was happy with the finished product. It will be released once the remaining content has been filled in and a suitable domain name has been purchased.

InstaWatch

User Experience Design

InstaWatch is a wholly original TV/Movie streaming application which focuses on the social aspect of watching TV shows/movies at the same time as friends within the application.

The original idea came out of an ideation session where the group found a problem with modern streaming services. We wanted to create an application where all streaming services could be found in one application, reducing the amount of time it takes users to find their desired show.

Upon conducting further user research, the group learned that the target audience would instead prefer using InstaWatch as a social streaming platform. This is where the original idea had shifted to instead focus more on the needs of the users.

Extensive competitor analysis, detailed personas and testing paper wireframes, axure prototypes and eventually a completed prototype (made in Invision) were constructed with constant iterations along the way to produce the best possible product.

Overall grade: 75%

Web Programming

Considering that JavaScript is the most in-demand programming language right now, these applications are a good way to demonstrate my understanding of the language.

These applications include different elements such as encryption/decryption, an order form and an example of a pub menu which incorporates AJAX and JSON.

Order Form

The customer order form demonstrates an understanding of calculations through JavaScript and uses different form elements to influence the final price of the completed form.

Once the form has been filled out by the user, a message at the bottom of the page would appear with all the final deductions/additions.

Overall grade: 78%

Encryptor

This application would take any message from the user and encrypt it into a new combination of letters, dependent on their key.

The user then has the option to copy and paste the encrypted message provided and input into a decryptor. When run, the initial user input would be returned.

Overall grade: 85%

Pub Menu

This pub menu demonstrates the use of AJAX and JSON. Features include an externally stored JSON file which is used to change the pub menu depending on which day the user chooses.

It also features a picture carousel towards the bottom right of the page which changes photo every 4 seconds to add a bit more life to the site.

Time constraints meant that the search filter was not able to be completed in time.

Overall grade: 67%
