Buy-ecars (Coming soon)

Website (self-initiated project)

Buy-ecars is the most ambitious project I've worked on yet. Myself and a friend have dedicated an immense amount of our personal time (spanning over a year) to build the UK's first electric-only car marketplace. The goal of buy-ecars is provide a platform which allows both independent sellers and traders to buy and sell electric cars.

The following features are available for it's users:

  • Fully functional search
  • Login / Register functionality
  • It's very own messaging service
  • Ability to upload your own ads (incl. images)
  • Automatic email service
  • Sleak and fully responsive modern design

We are aiming for the project to be released around the beginning of the new year. The aim is to provide an MVP (Minimum Viable Product) for it's launch and then to continuously incrementally improve it as time goes on.

The site has been built using React for the front-end and Java for the back-end.

InstaWatch

User Experience Design (University Group Project)

InstaWatch is a wholly original TV/Movie streaming application which focuses on the social aspect of watching TV shows/movies at the same time as friends within the application.

The original idea came out of an ideation session where the group found a problem with modern streaming services. We wanted to create an application where all streaming services could be found in one application, reducing the amount of time it takes users to find their desired show.

Upon conducting further user research, the group learned that the target audience would instead prefer using InstaWatch as a social streaming platform. This is where the original idea had shifted to instead focus more on the needs of the users.

Extensive competitor analysis, detailed personas and testing paper wireframes, axure prototypes and eventually a completed prototype (made in Invision) were constructed with constant iterations along the way to produce the best possible product.
